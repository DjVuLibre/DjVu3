DjVuFileCache.h

Simple DjVuFile caching class.
class DjVuFileCache: public GPEnabled
DjVuFileCache is a simple list of DjVuFile instances.
Files "DjVuFileCache.h" and "DjVuFileCache.cpp" implement a simple caching mechanism for keeping a given number of DjVuFile instances alive. The cache estimates the size of its elements and gets rid of the oldest ones when necessary.

See DjVuFileCache for details.
