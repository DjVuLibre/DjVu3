DjVu Navigation Directory.
Files "DjVuNavDir.h" and "DjVuNavDir.cpp" contain implementation of the multipage DjVu navigation directory. This directory lists all the pages, that a given document is composed of. The navigation (switching from page to page in the plugin) is not possible before this directory is decoded.

Refer to the DjVuNavDir class description for greater details.
