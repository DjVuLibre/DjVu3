DJVUEXTERNCAPI(void DjVuMessage_LookUp( char *msg_buffer, const unsigned int buffer_size, const char *message ))

A C function to perform a message lookup.

Documentation

A C function to perform a message lookup. Arguments are a buffer to received the translated message, a buffer size (bytes), and a message_list. The translated result is returned in msg_buffer encoded in UTF-8. In case of error, msg_buffer is empty (i.e., msg_buffer[0] == '\0').
