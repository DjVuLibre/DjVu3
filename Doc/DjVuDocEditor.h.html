DjVuDocEditor.h

DjVu document editor class.
class DjVuDocEditor: public DjVuDocument
DjVuDocEditor is an extension of DjVuDocument class with additional capabilities for editing the document contents.
Files "DjVuDocEditor.h" and "DjVuDocEditor.cpp" contain extension of DjVuDocument class, which can create and modify existing DjVu document, generate thumbnails, etc. It does not do compression though.
