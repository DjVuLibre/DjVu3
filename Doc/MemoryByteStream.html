Documentation

ByteStream interface managing a memory buffer. Class ByteStream::Memory manages a dynamically resizable buffer from which data can be read or written. The buffer itself is organized as an array of blocks of 4096 bytes.
MemoryByteStream()
Constructs an empty ByteStream::Memory. The buffer is initially empty. You must first use function write to store data into the buffer, use function seek to rewind the current position, and function read to read the data back.

MemoryByteStream(const void *buffer, size_t size)
Constructs a Memory by copying initial data. The Memory buffer is initialized with size bytes copied from the memory area pointed to by buffer.

MemoryByteStream(const char *buffer)
Constructs a Memory by copying an initial string. The Memory buffer is initialized with the null terminated string buffer.

void empty(void)
Erases everything in the Memory. The current location is reset to zero.
