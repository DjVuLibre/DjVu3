Documentation

This class contains some trivial annotations of the page or of the document such as page border color, page alignment, initial zoom and display mode, hyperlinks and highlighted areas. All this information is put inside a textual chunk ANTa in pseudo-lisp format. Decoding and encoding are normally done by decode() and encode() functions.
This class has no child classes.
